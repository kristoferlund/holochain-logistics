# Logistics

**WORK IN PROGRESS**

### Test

```
cd dna-src/test
hc test
```

### Holochain DNA

Building the DNA also requires that the holochain developer CLI, `hc`, is installed. Once you have `hc` installed, you can run the helper script

```
npm run hc:build
npm run hc:start
```

### UI

```
cd ui-src
yarn
npm start
```
